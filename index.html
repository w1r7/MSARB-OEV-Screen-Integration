MAX-SAFE DM1 Calculator and Live Trace Simulator

Single-packet DM1 only: one active DTC is transmitted per DM1 frame for this customer integration.

Simple MAX-SAFE Fault Selector

How to use this

Use this like a simple fault translator. First choose what failed, then choose how it failed. The tool shows the exact CAN message MAX-SAFE would transmit.

SPN means the faulty component or signal. FMI means how it failed. For this customer, the trace uses single-packet DM1 only, so if several faults are added the live trace sends them one-by-one, one fault every second.

Active faults

Single-Packet DM1 Outcome

The live trace sends one single-packet DM1 frame per second. If multiple faults are active, it rotates through them one-by-one. You can add or remove faults while the trace is running.

Customer restriction: this version does not generate BAM / multi-packet DM1. If multiple faults are active, MAX-SAFE rotates them one-by-one as single-packet DM1 frames, one fault per second. No BAM / multi-packet DM1 is generated.
