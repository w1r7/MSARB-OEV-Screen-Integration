MAX-SAFE DM1 Calculator and Live Trace Simulator

Single-packet DM1 only: one active DTC is transmitted per DM1 frame for this customer integration.

Simple MAX-SAFE Fault Selector

How to use this

Use this like a simple fault translator. Choose a fault from the list, then click Add selected fault. The tool adds it to the active fault stack and creates the DM1 message accordingly.

SPN means the faulty component or signal. FMI means how it failed. For this customer, the trace uses single-packet DM1 only. If several faults are active, the live trace rotates through them one-by-one, one fault every second. When no faults are active, MAX-SAFE sends the no-fault DM1 frame.

Live Trace Simulator

The live trace sends one single-packet DM1 frame per second. If multiple faults are active, it rotates through them one-by-one. You can add or remove faults while the trace is running.

DM1 data bytes

ByteContains
Data[0]Lamp status, four 2-bit fields
Data[1]Flash status, four 2-bit fields
Data[2]SPN bits 0-7
Data[3]SPN bits 8-15
Data[4]SPN bits 16-18 in bits 0-2, FMI in bits 3-7
Data[5]OC in bits 0-6, CM in bit 7
Data[6-7]Unused/padding for one DTC
Customer restriction: this version does not generate BAM / multi-packet DM1. If multiple faults are active, MAX-SAFE rotates them one-by-one as single-packet DM1 frames, one fault per second. No BAM / multi-packet DM1 is generated.
